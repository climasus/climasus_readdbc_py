# Instalação

## Requisitos

- Python ≥ 3.10
- pandas ≥ 2.0

## Via pip

```bash
pip install climasus_readdbc_py
```

## Em projetos com `pyproject.toml`

```toml
[project]
dependencies = [
    "climasus_readdbc_py>=0.1.0",
]
```

## Importação

O pacote é instalado como `climasus_readdbc_py` no PyPI, mas importado como `readdbc`:

```python
import readdbc
```
